# LancheFlow

O LancheFlow é um sistema desenvolvido para auxiliar no gerenciamento de uma lanchonete, facilitando o controle de pedidos, produtos e atendimento aos clientes.

O objetivo do projeto é resolver problemas comuns encontrados no atendimento manual, como demora nos pedidos, dificuldade no controle de estoque e organização das comandas. Com o sistema, será possível registrar pedidos de forma rápida, acompanhar vendas e melhorar a experiência dos clientes.

Funcionalidades Iniciais
Cadastro de produtos
Controle de pedidos
Gerenciamento de clientes
Controle de estoque
Tela de login para funcionários
Painel administrativo
